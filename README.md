

# 🎬 MovieHub – Movie Information Webpage

## 📌 Project Overview

**MovieHub** is a simple movie information webpage developed using **HTML, CSS, and Bootstrap (for slider only)**.


The website includes:

* Title and Logo
* Navigation Bar
* Introduction Section
* Internal and External Links
* Image Slider
* Information Table
* Blockquote Section
* Footer

---

## 🎯 Assignment Objective

To apply:

* HTML structure
* CSS styling
* Links (Internal & External)
* Tables
* Image Slider using Bootstrap
* Basic layout and UI design principles

---

## 🛠 Technologies Used

* **HTML5**
* **CSS3**
* **Bootstrap 5 (CDN - Only for Image Slider)**

## ✨ Features

### 1️⃣ Title Section

* Displays project name at the top.
* Styled with custom CSS.

### 2️⃣ Navbar

* Contains:

  * Logo space
  * Internal links (About Us, Contact Us)
  * External links (TMDB, IMDb)

### 3️⃣ Introduction Section

* Short paragraph explaining the system.

### 4️⃣ Image Slider

* Bootstrap carousel
* Contains 3–4 images
* Navigation arrows enabled

### 5️⃣ Information Table

* Displays movie categories
* Includes category name, description, and popularity

### 6️⃣ Blockquote

* A relevant slogan about cinema

### 7️⃣ Footer

* Copyright information
